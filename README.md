# Bloxdkeys
Bloxd keys is a script that is designed ofr users to be able to download and upload keybinds
This can be used to swap between different keybinds easily.



## How to use it

### Installing

1. Make sure you have [Tampermonkey](https://www.tampermonkey.net/) installed in your browser.  
2. Create a **new userscript** in Tampermonkey.  
3. Copy and paste the script into the editor.  
4. Open [Bloxd.io](https://bloxd.io) and go to **controls**.
